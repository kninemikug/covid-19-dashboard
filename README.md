# COVID-19 Dashboard

Streamlit 기반의 코로나19 현황 대시보드 프로젝트입니다.

## 설치 방법 (Installation)

이 프로젝트는 Python 3.8+ 환경에서 실행하는 것을 권장합니다.

### 1. 가상환경 생성 (선택 사항)
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
# venv\Scripts\activate  # Windows
```

### 2. 패키지 설치
`requirements.txt`에 명시된 필수 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

## 실행 방법 (Usage)
```bash
streamlit run app.py
```
